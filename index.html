About The OOTP Uniform Maker
So what is this thing?

This is a utility designed to make it easy to create custom uniforms for Out Of The Park Baseball 23 and newer. If you don't at least have OOTP23, then this is really of little use to you.

What is Out Of The Park Baseball?

Out Of The Park Baseball is an award winning baseball management simulation published by OOTP Developments. For more information about OOTP, as well as the other games they make, visit their site.

How did you do this?

The Uniform Maker is built using Node.js and Electron. So it's written entirely in HTML and JavaScript like pretty much any website.

Who is responsible for this thing?
That would be me. My name is Eriq Jaffe, and I've been playing OOTP Baseball since 2004. Feel free to shoot me an email with any questions or comments. You can also view the source code for this on GitHub.

Some of the textures and shapes are taken directly from OOTP, but a lot of them are based on the work of Gern44, silvam14, txranger, Deft, NoPepper, Reds1, davwms and probably several others (my record keeping isn't all that great, and I'm not 100% sure where every piece came from - if I'm using one of your components and I forgot to give you credit, thanks!).

I'd also like to give an additional shout out to Conrad Burry, who created some of the default fonts.
